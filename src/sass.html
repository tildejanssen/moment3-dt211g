SASS

SASS tycker jag har förenklat processen med användingen av CSS som vanligtvis kan ta mycket tid då man inte har variabler att använda utan måste göra allt en åt gången. Color functions som darken och lighten har gjort det lättare att fram ett snyggt och passande färgschema för hela webbplatsen. Koden blir väldigt mycket lättare att läsa då allt är tydligt och strukturerat, partials bidrar även till detta. Eftersom SASS är ett nytt kodspråk att lära sig så tog det lite tid i början att förstå och komma ihåg alla nya funktioner som går att använda. Det positiva med SASS är att nesting gör koden mer läsbar, variabler går att återanvända och sparar tid om man vill göra stora förändringar, mixins gör att man kan återanvända kod så att man inte behöver upprepa sig. Det negativa med SASS skulle kunna vara att det tar lite extra tid att lära sig eftersom det är ett annat typ av kodspråk, jämfört med CSS så måste SASS konverteras till CSS för att kunna användas och visas i webbläsaren, vilket då lägger till lite fler steg i utvecklingsprocessen. Överanvänder man funktioner, mixins och nesting kan det leda till för komplexa och invecklade stilmallar som i sin tur dessutom kan påverka laddningstiden hos webbläsaren. Någonting jag personligen märkt med SASS är att "hosten" lätt kraschar och måste omstartas.
